Пакет вагонов РЖД. Версия 3.0
__________________________
Поставляется вместе с оригинальным дистрибутивом RRS. Однако могут выходить обновления и патчи, следите за информацией!
В комплект состава входят следующие модели:
- pass_rzd-11100, купейный вагон Аммендорф с КУ;
- pass_rzd-13819, купейный вагон Аммендорф с КУ;
- pass_rzd-16733, штабной купейный вагон Аммендорф с КУ прошедший КВР с радиокупе и купе для ЛОВ;
- pass_rzd-17669, купейный вагон-бар Гёрлиц;
- pass_rzd-65361, вагон-ресторан;
- pass_rzd-25924, плацкартный вагон ТВЗ с КУ(61-826) после КВР;
Вагоны приписаны к Московской железной дороге.
